## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/harshjha11/python_learning.git
```

### Navigate into the project

```bash
cd python_learning
```

### Run any Python file

```bash
python filename.py
```

or

```bash
python3 filename.py
```
